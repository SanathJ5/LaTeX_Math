Natural Math --> Python/MATLAB/LaTeX

Enter the math expressions in the natural way below

  • DESMOS INSPIRED
  • \sqrt for square root
  • \pi for pi
  • Use parentheses for arguments
  • sin x --> WRONG, sin(x) --> RIGHT. Same for all trignometric and logarithmic functions
  • Currently cosec, sec and cot are not supported
  • sin^2(x) --> WRONG, (sin(x))^2 --> RIGHT. Same for all trignometric functions
  • Single-letter mode enables auto multiplication
  • If possible use Multi Letter Variable mode for better stability
